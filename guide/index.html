Index

Navigate

The Index is a navigational sidebar panel (240px wide) that provides a structured overview of all notes in the current project. It appears on the right side of the screen.

How to Open

  • Press Ctrl+K → select Index
  • Click the Index toggle button in the status bar
  • Close with the X button in the panel header, or by opening a different panel

Tiling / Graph View — Category Groups

In Tiling or Graph view, the Index header reads "Canvas Index". Notes are grouped by category (e.g. "Philosophy", "Technology", "Tasks"). Within each category, every note is listed as a clickable row showing:

  • A type icon — checkmark for tasks, sparkle for theses, question mark for questions, etc.
  • A preview of the first 35 characters of the note text

Kanban View — Column Directory

In Kanban view, the header reads "Board Index". Instead of listing individual notes, it shows a column directory — each kanban column (by content type) — so you can jump directly to a specific board column.

Navigation

Hover over a note entry → the corresponding tile on the canvas is highlighted.
Click a note entry → smooth-scroll to that tile and briefly highlight it (fades after 1.5 seconds).
In Kanban mode, clicking a column name smooth-scrolls horizontally to that column.

This makes the Index the primary tool for quickly locating and jumping to a specific note when your canvas grows large.
