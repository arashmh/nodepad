New Project

Navigate

Creates a brand-new, empty project (called a "Space" in the UI) and immediately switches to it.

How to Trigger

  • Press Ctrl+K → select New Project
  • Open the sidebar → click New Space in the footer

What Happens

A single-click action — no wizard, no template picker, no prompt. The new project is created instantly and becomes the active canvas.

The new project starts with:

  • Name: "New Space"
  • Notes: empty (no blocks)
  • Collapsed state: nothing collapsed
  • Ghost notes: none

After Creation

The sidebar opens automatically so you can see the new project in the list. To rename it, hover over the project entry and click the pencil icon.

The project is appended to the end of the project list. A unique random ID is generated to prevent collisions.

First-Time Experience

On a fresh install with no saved projects, the app creates one default project named "✨ New Research" — structurally identical to a manually created space, just with a friendlier name.
