Clear Canvas

Action

Removes all notes from the current project. The project itself remains — only the notes inside it are cleared.

How to Trigger

  • Press Ctrl+K → select Clear canvas

What Gets Cleared

  • All blocks (notes) → set to empty array
  • All collapsed states → reset

What's Preserved

  • The project itself — name, ID, and position in the list remain intact
  • Ghost notes — the synthesis panel is untouched

No Confirmation Dialog

The action executes immediately — there's no "Are you sure?" prompt. However, it is fully undoable.

Undo

Before clearing, the current notes are pushed onto an undo history stack (up to 20 snapshots). Press Ctrl+Z (or Cmd+Z on Mac) to restore everything. A brief toast notification confirms:

  • "↩ Undone" — notes successfully restored
  • "Nothing to undo" — the undo stack is empty
