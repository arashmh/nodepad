Copy Markdown

Action

Copies the full markdown export of the current project to your system clipboard — ready to paste into any markdown editor, document, or chat.

How to Trigger

  • Press Ctrl+K → select Copy markdown

What You Get

Functionally identical to Export Markdown — the same rich document with YAML front matter, table of contents, per-type sections, confidence bars, and full annotations. The only difference is the delivery method: clipboard instead of file download.

Content Includes

  • YAML front matter (title, date, node count, types)
  • Overview table with type emoji and counts
  • Pinned nodes callout
  • Table of contents
  • All note sections with type-specific formatting
  • Footer with attribution

Use Cases

  • Paste into an LLM chat — the structured format gives AI models excellent context about your research
  • Paste into Obsidian / Notion — renders cleanly in most markdown editors
  • Share in a message — quick way to share your research canvas without sending a file
